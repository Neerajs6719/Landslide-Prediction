About the Landslide Risk Prediction System

Project Overview

This system was developed to help predict landslide risks using machine learning algorithms. It analyzes environmental factors that contribute to landslides and provides risk assessments.

The model was trained on historical landslide data with the following features:

  • Temperature
  • Humidity
  • Precipitation
  • Soil Moisture
  • Elevation

Technology Stack

  • Machine Learning: Random Forest Classifier (Scikit-learn)
  • Backend: Python, Flask
  • Frontend: HTML5, CSS3, Bootstrap 5
  • Visualization: Matplotlib

Team

This project was developed by a team dedicated to using technology for natural disaster prevention.
